# SpocAutoPlayer

> 北航 SPOC 平台视频进度模拟器 —— 适用于网络不佳导致视频加载缓慢的情况

**SpocAutoPlayer** 通过直接发送 API 请求更新观看进度，无需实际播放视频。

## 功能特点

- **快速刷课** - 无需等待视频加载和播放
- **两种模式** - 渐进式模拟 / 一键完成
- **可视化进度** - 实时显示进度条和预计完成时间
- **灵活配置** - 自定义播放速度和心跳间隔
- **全自动化** - 自动拦截请求、自动获取视频时长

## 使用方法

1. 打开 SPOC 课程视频页面
2. 按 `F12` 打开开发者工具
3. 切换到 **Console (控制台)** 标签
4. 复制 `console_auto.js` 的全部代码并粘贴运行
5. 阅读并同意用户协议
6. 点击视频的 **"查看"** 按钮，脚本自动拦截请求
7. 确认配置后点击 **"开始"** 按钮

## 实现原理

本工具通过拦截浏览器发送到 SPOC 服务器的 XHR 请求，自动提取课程参数和认证 Token。然后模拟视频播放过程，定时向服务器发送进度更新请求，实现无需实际播放即可完成视频观看进度的效果。

## 参数说明

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 视频时长 | 视频的总时长（秒），脚本会尝试自动获取 | 自动获取 |
| 一键完成 | 直接将进度标记为100%，速度快但风险高 | 关闭 |
| 播放倍速 | 模拟的播放速度倍数 | 10-20x |
| 心跳间隔 | 向服务器发送进度更新请求的时间间隔 | 1-2 秒 |

**推荐配置**: 播放倍速 10-20x，心跳间隔 1-2秒。以上配置可在约30秒内完成5分钟视频，同时降低被检测风险。
